DEVELOPMENT
===========

Setup dev environment:
  python -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
  pip install pytest black flake8 mypy

Run tests:
  pytest tests/ -v --cov=src

Format code:
  black src/ tests/

Check code:
  flake8 src/
  mypy src/

Build Docker:
  docker build -t ultravox-mcp .
